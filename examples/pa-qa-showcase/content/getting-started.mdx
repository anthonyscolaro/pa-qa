# Getting Started with PA-QA

Welcome to the PA-QA Testing Framework! This guide will help you quickly set up comprehensive testing for your web development projects.

## What is PA-QA?

PA-QA is a multi-agent testing framework that provides standardized testing templates, shared utilities, and best practices for modern web development. It supports React, WordPress, FastAPI, and more.

<CodeDemo 
  title="Quick Setup Example"
  language="bash"
  code={`# Clone a project template
git clone https://github.com/your-agency/pa-qa-templates.git
cd pa-qa-templates/project-types/web-apps/react

# Install dependencies
npm install

# Run all tests
npm run test:all

# View coverage report
npm run test:coverage`}
/>

## Key Features

- **Multi-Agent Workflows**: Generate complete test suites in minutes using parallel agent execution
- **Framework Support**: React, WordPress, FastAPI, and more
- **Docker-First**: Consistent testing environments across development and CI/CD
- **Allure Integration**: Centralized reporting dashboard
- **Modern Tooling**: Vitest, Playwright, PHPUnit 9.x, Pytest 7.x

### 2. Copy Template Files

Once you've selected your project type, copy the relevant template files:

<CodeDemo
  title="React Project Setup"
  language="bash"
  code={`# Copy React testing template
cp -r pa-qa/project-types/web-apps/react/tests/ ./
cp -r pa-qa/project-types/web-apps/react/configs/ ./
cp pa-qa/project-types/web-apps/react/package.json ./package-pa-qa.json

# Install testing dependencies
npm install --save-dev $(cat package-pa-qa.json | jq -r '.devDependencies | keys[]')`}
  runnable={false}
/>
